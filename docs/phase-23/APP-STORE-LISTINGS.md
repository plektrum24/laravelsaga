## Analytics Setup

### Firebase Analytics

**Setup Steps:**

1. **Create Firebase Project:**
   - Go to https://console.firebase.google.com
   - Create new project: "SAGA POS Mobile"
   - Add iOS app (bundle ID: com.sagaposo.mobileapp)
   - Add Android app (package: com.sagaposo.mobileapp)
   - Download config files

2. **Install Firebase SDK:**
```bash
npm install @react-native-firebase/app
npm install @react-native-firebase/analytics
npm install @react-native-firebase/crashlytics
```

3. **Configure in app.json:**
```json
{
  "expo": {
    "ios": {
      "googleServicesFile": "./GoogleService-Info.plist"
    },
    "android": {
      "googleServicesFile": "./google-services.json"
    }
  }
}
```

4. **Key Events to Track:**
```javascript
// Screen views
await analytics().logScreenView({
  screen_name: 'Home',
  screen_class: 'HomeScreen',
});

// Product interactions
await analytics().logViewItem({
  item_id: product.id,
  item_name: product.name,
  value: product.price,
});

// Add to cart
await analytics().logAddToCart({
  item_id: product.id,
  item_name: product.name,
  value: product.price,
  quantity: 1,
});

// Purchase
await analytics().logPurchase({
  transaction_id: order.id,
  value: order.total,
  currency: 'IDR',
  items: orderItems,
});

// Loyalty actions
await analytics().logEvent('earn_points', {
  points: 100,
  source: 'purchase',
});
```

### Crashlytics Setup

**Install:**
```bash
npm install @react-native-firebase/crashlytics
```

**Usage:**
```javascript
import crashlytics from '@react-native-firebase/crashlytics';

// Log custom keys
await crashlytics().setAttributes({
  user_tier: 'gold',
  app_version: '2.1.0',
});

// Log non-fatal errors
await crashlytics().recordError(error, {
  context: 'payment_flow',
});

// Log messages
crashlytics().log('User completed checkout');
```

### Performance Monitoring

**Install:**
```bash
npm install @react-native-firebase/perf
```

**Usage:**
```javascript
import perf from '@react-native-firebase/perf';

// Track custom traces
const trace = await perf().newTrace('product_search');
await trace.start();

// Your code here
await searchProducts(query);

await trace.stop();
const duration = await trace.getMetric('duration');
```
