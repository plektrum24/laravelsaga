## 3. API Endpoints (Backend)

Base URL: `/api`

### Authentication
- `POST /login` - Login user (Returns Sanctum Token)
- `POST /logout` - Logout (Revoke Token)
- `GET /user` - Get current user profile (& Role)

### Cash Register (Shift)
- `GET /cash-register/current` - Check status of current shift
- `POST /cash-register/open` - Open new shift (Modal Awal)
- `POST /cash-register/close` - Close shift (Laporan Akhir)
- `POST /cash-register/expense` - Record Expense (Uang Keluar)

### Products (Inventory)
- `GET /products` - List all products
- `POST /products` - Create product
- `GET /products/{id}` - Detail product
- `PUT /products/{id}` - Update product
- `DELETE /products/{id}` - Delete product
- `GET /products/categories` - List categories
